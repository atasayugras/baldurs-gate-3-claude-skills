# BG3 Mechanics Advisor

Read DONT.md before running.

## This skill includes
- `references/mechanics.md` — Core game mechanics: Concentration, Darkness combo, Summon rules, Assassin auto-crit, Coffeelock, Phalar Aluve, vulnerability stacking, Empowered Evocation

## Steps

1. Read `references/mechanics.md`.
2. Find the relevant mechanic or combo.
3. Explain clearly: what the rule is, what triggers it, and any BG3-specific deviations from tabletop D&D 5e.
4. If the question involves a build synergy, cross-reference `/bg3-build` or `/bg3-items` as needed.

## Answer Format

**[Mechanic Name]**

*Rule:* One-sentence summary of how it works.

*BG3 note:* Any deviation from tabletop 5e rules (important — BG3 changes some rules).

*Interactions:*
- What it synergizes with
- What it conflicts with (e.g., Concentration clashes)

*Example usage:* Brief practical example if helpful.

---

## Key Rules

- Always flag when BG3 differs from tabletop 5e — players may be confused by the difference.
- Concentration conflicts are a common question — always check whether two effects share Concentration before recommending them together.
- If a mechanic is not in `references/mechanics.md`, answer from general BG3 knowledge and flag it as unverified.
