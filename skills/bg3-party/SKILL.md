# BG3 Party Advisor

Read DONT.md before running.

## This skill includes
- `references/party-synergies.md` — Party archetypes, companion build recommendations, synergy patterns

## Steps

1. Read `references/party-synergies.md`.
2. Identify the user's main build (ask if not stated).
3. Recommend a party of 3 companions with specific builds and roles.
4. Explain the synergy — why this party works together.

## Answer Format

### Party for [Main Build]

**Main:** [Build Name] — Role

**Companion 1: [Name]**
- **Build:** Class A X / Class B Y
- **Role:** What they contribute
- **Why:** How they complement the Main

**Companion 2: [Name]**
- **Build:** ...
- **Role:** ...
- **Why:** ...

**Companion 3: [Name]**
- **Build:** ...
- **Role:** ...
- **Why:** ...

**Synergy Summary:** One paragraph on how this party plays together — what the typical combat loop looks like.

**Achievement Note:** If any companion choice affects achievements (e.g., Wyll for Loophole), flag it.

---

## Party Pillars

Every party needs these covered — check that the recommended party has all:
- [ ] Frontline / tank (someone who can take hits)
- [ ] Healing or sustain (or potions budget)
- [ ] AoE damage or control
- [ ] Utility (locks, perception, social skills outside combat)

If the Main covers one of these, note it.

## Companion Count

BG3 party cap is 4 (Main + 3). If the user wants to swap companions for companion quests, note who to bring for which quest and when.
